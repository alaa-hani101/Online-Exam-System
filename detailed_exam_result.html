Explain the difference between localStorage and sessionStorage in web applications.

localStorage and sessionStorage are both web storage options that allow data to be stored in the browser. The main difference is that localStorage persists even after the browser is closed while sessionStorage is cleared when the session ends.

Instructor Feedback:

Your answer correctly identified the primary difference in persistence but didn't mention other important distinctions such as storage limits (both typically 5MB per domain) and that sessionStorage is specific to a single window/tab while localStorage is shared across all windows/tabs from the same origin.

Describe the purpose of JavaScript frameworks like React, Angular, or Vue. Why are they useful in modern web development?

JavaScript frameworks like React, Angular, and Vue provide structured environments for building complex web applications. They offer component-based architecture that makes code more reusable and maintainable. These frameworks handle complex DOM manipulations efficiently through virtual DOM implementations, improve development speed with ready-to-use components, and enforce best practices. They also provide state management solutions which is crucial for large applications. Additionally, they support creating single-page applications (SPAs) that offer better user experiences with faster transitions between views.
